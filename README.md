## Proposta do projeto
Este projeto é uma iniciativa pessoal para estudar e replicar uma arquitetura corporativa distribuída. 

Este repositório contém a página e componentes do **Envio de arquivos para um bucket S3**, ele é um Micro Front End ou seja uma página individual.

**🔗 Ecossistema do Projeto (Outros Repositórios):**
- 🏠 [App Shell (Host)](https://github.com/thiagosilvaantenor/dados-app-shell)
- ⚙️ [BFF API & Cloud (Backend)](link_do_seu_repo_bff)
- 📊 [MFE Dashboard (Remote)](https://github.com/thiagosilvaantenor/dados-mfe-dashboard)
